Wolf Trap Aria Frequency List, 2013-2019

Wolf Trap Aria Frequency List (link to spreadsheet)

For a number of years, Wolf Trap Opera released a complete list of every aria offered throughout their audition process. This document compiles these lists, from 2013-2019, to create a full aria frequency list that serves as a rough guide of the most popular arias for young American singers. Singers, teachers, and coaches can use this list to get ideas for common (or uncommon) audition arias based on composer, voice type, language, and era. Wolf Trap no longer shares these lists, so (I believe) this is currently the only existing data source of its kind.

Full instructions can be found in the first tab of the spreadsheet. I’ve found it extremely helpful as reference material, and I hope you’ll find the same! If you have corrections, questions, or suggestions about how to improve the sheet, email me at jonathan.heaney@gmail.com.
